Bem-vindo ao sistema de gerenciamento de imóveis

Rotas disponíveis

  • Listar todos os imóveis
  • Listar imóvel por ID
  • Listar imóveis por tipo
  • Listar imóveis por cidade
  • Adicionar novo imóvel
  • Atualizar imóvel
  • Remover imóvel

Sobre o Projeto

Esta API foi desenvolvida utilizando Flask, com persistência em MySQL (Aiven), seguindo os princípios de TDD. O deploy foi realizado em um servidor AWS EC2.
